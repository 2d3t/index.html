Стеганография в PNG

Встраивайте одно изображение в другое и извлекайте скрытые изображения с помощью методов стеганографии

Как это работает

Это приложение использует метод LSB (Least Significant Bit - наименее значащий бит) для скрытия одного изображения внутри другого. При кодировании младшие биты основного изображения заменяются битами скрываемого изображения. При декодировании эти биты извлекаются для восстановления скрытого изображения.

Инструкция по использованию

  1. Для кодирования: загрузите основное изображение (контейнер) и изображение, которое хотите скрыть. Выберите битовую глубину (чем больше бит, тем больше данных можно скрыть, но тем заметнее будут изменения в основном изображении). Нажмите "Закодировать изображение".
  2. Для декодирования: загрузите изображение со скрытыми данными. Выберите ту же битовую глубину, что использовалась при кодировании. Нажмите "Декодировать изображение".
  3. Сохраните результат, нажав правой кнопкой мыши на изображении и выбрав "Сохранить как".
  4. Рекомендация: используйте PNG изображения для наилучшего результата, так как формат PNG не использует сжатие с потерями.
