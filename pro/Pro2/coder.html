Как работает это приложение

Это приложение использует метод стеганографии для скрытия текста в изображениях путем изменения значений RGB пикселей:

  • Кодирование: Текст разбивается на символы, каждый символ преобразуется в ASCII код, который затем кодируется в младших битах значений RGB пикселей.
  • RGB смещение: Определяет, сколько бит используется для кодирования (от 1 до 3). Меньшие значения делают изменения менее заметными.
  • Маркер окончания: В конец данных добавляется специальная последовательность для определения конца текста.
  • Декодирование: Извлекаются младшие биты из значений RGB, собираются в ASCII коды и преобразуются обратно в текст.

Важно: Используйте одинаковое значение смещения для кодирования и декодирования. Изображение в формате JPG может вносить небольшие искажения из-за сжатия.
